# axshb's dotfiles
### color scheme based on cherry midnight

### main packages:
- btop (system monitor)
- kanshi (display management)
- kitty (terminal emulator)
- labwc (a wlroots stacking compositor)
- mako (notification daemon)
- rofi (launcher)
- waybar (bar)
- yazi (file manager)
- fastfetch (fetch)

the other packages (system services, apps, etc) are in configuration.nix and home.nix for system and user packages respectively.
